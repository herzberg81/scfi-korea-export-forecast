### 1.2 Python环境安装

#### 1.2.1 Windows系统安装Python

**步骤一：下载Python**

1. 访问Python官方网站：https://www.python.org/downloads/
2. 选择Python 3.9或更高版本的Windows安装包
3. 下载64位安装包（推荐）

**步骤二：安装Python**

1. 双击运行下载的安装包
2. **重要**：勾选"Add Python to PATH"选项
3. 选择"Install Now"进行安装
4. 等待安装完成

**步骤三：验证安装**

打开命令提示符（CMD），输入以下命令：

```bash
python --version
```

预期输出：Python 3.9.x 或更高版本

#### 1.2.2 Linux系统安装Python

```bash

sudo apt install python3.9
sudo apt install python3-pip
```


#### 4.2.2 模型训练

```python
# 运行模型训练脚本
python -c "
from data.DataReader import DataReader
from data.DataFusionPreprocessor import DataFusionPreprocessor
from model.SCFIForecastModel import SCFIForecastModel

# 读取数据
reader = DataReader('../data_source/上海出口集装箱运价指数.xlsx', '../data_source/韩国出口指数.xlsx')
scfi_df, korea_df = reader.get_original_two_index()

# 预处理
preprocessor = DataFusionPreprocessor(scfi_df, korea_df)
fusion_df = preprocessor.merge_two_index()
fusion_df = preprocessor.clean_data(fusion_df)
feature_df = preprocessor.build_lag_feature(fusion_df)
X_train, X_val, y_train, y_val = preprocessor.split_dataset(feature_df)

# 训练模型
model = SCFIForecastModel(model_type='random_forest')
result = model.train(X_train, y_train, X_val, y_val)
print(f'模型训练完成，R²={result[\"metrics\"][\"r2\"]}')

# 保存模型
model.save_model('my_model.pkl')
print('模型已保存至 models_save/my_model.pkl')
"
```

## 六、报错排查

### 6.1 常见错误及解决方案

#### 错误 E01：文件不存在

**错误信息**：
```
FileNotFoundError: [Errno 2] No such file or directory: 'data_source/上海出口集装箱运价指数.xlsx'
```

**原因分析**：
- 数据文件未放置在data_source目录
- 文件路径配置错误
- 文件名不符合规范

**解决方案**：

1. 检查data_source目录是否存在数据文件
   ```bash
   # Windows
   dir data_source

   # Linux/macOS
   ls data_source
   ```

2. 确保文件名正确：
   - SCFI文件名：上海出口集装箱运价指数.xlsx
   - 韩国出口指数文件名：韩国出口指数.xlsx

3. 如果文件在其他位置，复制到data_source目录
   ```bash
   # 复制文件到data_source目录
   copy 你的数据文件路径\*.xlsx data_source\
   ```

#### 错误 E02：依赖包未安装

**错误信息**：
```
ModuleNotFoundError: No module named 'pandas'
```

**原因分析**：
- 未安装所需依赖包
- 虚拟环境未激活

**解决方案**：

1. 检查依赖包是否安装
   ```bash
   pip list
   ```

2. 安装缺失的依赖包
   ```bash
   pip install -r requirements.txt
   ```

3. 确保虚拟环境已激活
   ```bash
   # Windows
   venv\Scripts\activate

   # Linux/macOS
   source venv/bin/activate
   ```

#### 错误 E03：Python版本不兼容

**错误信息**：
```
SyntaxError: invalid syntax
```

**原因分析**：
- Python版本低于3.9
- 代码使用了高版本特性

**解决方案**：

1. 检查Python版本
   ```bash
   python --version
   ```

2. 如果版本低于3.9，升级Python
   - 下载Python 3.9或更高版本
   - 安装后重新运行程序

#### 错误 E04：数据格式错误

**错误信息**：
```
ValueError: could not convert string to float
```

**原因分析**：
- 数据文件包含非数值内容
- 日期格式不正确
- 数据列缺失

**解决方案**：

1. 检查Excel文件数据格式
   - 打开Excel文件检查数据
   - 确保数值列为数字类型
   - 确保日期列为日期格式

2. 修正数据格式
   - 将非数值内容删除或替换
   - 确保日期格式为YYYY-MM-DD

3. 确保所有必需字段存在
   - SCFI：日期 + 9个航线字段
   - 韩国出口指数：日期 + 6个出口指标字段

#### 错误 E05：数据量不足

**错误信息**：
```
ValueError: Found array with 0 sample(s) while a minimum of 1 is required
```

**原因分析**：
- 数据文件为空
- 数据清洗后剩余记录太少
- 时序不匹配导致大量数据被剔除

**解决方案**：

1. 检查数据文件是否有足够数据
   - 数据量建议≥50条

2. 检查两数据源日期是否匹配
   - 确保两数据覆盖相同时间范围

3. 减少数据清洗强度
   - 修改clean_data参数，放宽异常值阈值

#### 错误 E06：模型训练失败

**错误信息**：
```
RuntimeError: Model training failed
```

**原因分析**：
- 特征数据包含NaN值
- 标签数据格式不正确
- 模型参数配置错误

**解决方案**：

1. 检查特征数据完整性
   ```python
   # 检查NaN值
   print(X_train.isnull().sum())
   ```

2. 检查标签数据
   ```python
   print(y_train.dtype)
   ```

3. 检查模型参数配置
   - 查看train_config.py参数是否合理

#### 错误 E07：图表中文显示乱码

**错误信息**：
图表中的中文显示为方框或乱码

**原因分析**：

```python
model_types = ['linear', 'random_forest', 'xgboost']

    model = SCFIForecastModel(model_type=model_type)
    result = model.train(X_train, y_train, X_val, y_val)
```
