## 一、系统架构概述

### 1.1 设计原则

本系统采用分层面向对象架构设计，遵循以下设计原则：

1. **高内聚低耦合**：各模块独立封装，通过标准化接口交互
2. **单一职责**：每个类和模块只负责一个明确的功能
3. **开放封闭**：对扩展开放，对修改封闭
4. **依赖倒置**：高层模块不依赖低层模块，都依赖抽象接口

### 1.2 架构特点

- 面向对象设计，模块独立可替换
- 五层分层架构，职责清晰
- 标准化参数交互，耦合度低
- 支持数据源、模型、特征灵活扩展

### 2.2 各层详细说明

#### 第一层：数据接入层（DataReader）

**职责**：统一读取两类Excel数据源，提供标准化数据输出接口

**核心类**：DataReader

**主要功能**：
- 读取SCFI上海集装箱运价指数Excel文件
- 读取韩国出口先行指数Excel文件
- 自动跳过标题行（SCFI跳过1行，韩国出口指数跳过2行）
- 解析日期字段为标准格式
- 验证数据完整性

**输入**：
- SCFI Excel文件路径
- 韩国出口指数Excel文件路径

**输出**：
- SCFI原始DataFrame
- 韩国出口指数原始DataFrame

**数据流描述**：
```
Excel文件 → DataReader.read_scfi_excel() → SCFI DataFrame
Excel文件 → DataReader.read_korea_export_excel() → 韩国出口指数 DataFrame
```

---

#### 第二层：时序融合预处理层（DataFusionPreprocessor）

**职责**：解决两份数据时间轴对齐问题，构造滞后先行特征

**核心类**：DataFusionPreprocessor

**主要功能**：
- 按日期主键内连接合并两份指数数据
- 缺失值处理和异常值剔除
- 构造韩国出口指数滞后1-4周特征（核心创新）
- 构造SCFI滞后1-4周特征
- 划分训练集和验证集（按时间切分）

**输入**：
- SCFI DataFrame
- 韩国出口指数DataFrame

**输出**：
- 融合后的完整特征矩阵X
- 预测目标Y（下周SCFI）
- 训练集和验证集划分

**数据流描述**：
```
SCFI DataFrame + 韩国出口指数 DataFrame
    ↓
merge_two_index() → 按日期对齐合并
    ↓
clean_data() → 数据清洗
    ↓
build_lag_feature() → 构造滞后特征（韩国出口指数lag1-4, SCFI lag1-4）
    ↓
split_dataset() → 划分训练集(80%) / 验证集(20%)
    ↓
输出: 特征矩阵X, 目标Y
```

---

#### 第三层：预测模型层（SCFIForecastModel）

**职责**：训练预测模型，提供下周运价预测功能

**核心类**：SCFIForecastModel

**主要功能**：
- 多模型训练（线性回归、随机森林、XGBoost、ARIMA）
- 模型持久化保存和加载
- 输入当周数据预测下周SCFI
- 支持多航线预测

**输入**：
- 训练特征X_train
- 训练标签Y_train
- 模型类型配置

**输出**：
- 训练完成的预测模型
- 下周SCFI预测值

**数据流描述**：
```
特征矩阵X + 目标Y
    ↓
train() → 模型训练（支持多种算法）
    ↓
save_model() → 模型持久化保存(.pkl)
    ↓
输入当周数据 → predict_next_scfi() → 下周SCFI预测值
```

---

#### 第四层：结果分析层（ResultAnalyzer）

**职责**：评估模型精度，分析特征贡献度

**核心类**：ResultAnalyzer

**主要功能**：
- 计算R²、MAE、RMSE等精度指标
- 计算韩国出口指数与SCFI相关系数
- 输出特征重要性排序
- 量化先行指标预测价值

**输入**：
- 真实值Y_true
- 预测值Y_pred
- 训练模型对象

**输出**：
- 精度评估指标字典
- 相关系数
- 特征重要性排序

**数据流描述**：
```
真实值 + 预测值 + 模型
    ↓
calc_evaluation_metrics() → R², MAE, RMSE
    ↓
calc_correlation() → 韩国出口指数与SCFI相关系数
    ↓
get_feature_importance() → 特征重要性排序
```

---

#### 第五层：结果导出展示层（ResultExporter）

**职责**：生成可视化图表，导出预测报表

**核心类**：ResultExporter

**主要功能**：
- 绘制真实值vs预测值对比折线图
- 绘制相关性散点图
- 绘制特征重要性柱状图
- 绘制各航线运价趋势图
- 导出Excel周度预测报告
- 导出TXT模型评估日志

**输入**：
- 预测结果数据
- 评估指标
- 特征重要性数据

**输出**：
- PNG可视化图表
- Excel预测报告
- TXT评估日志

**数据流描述**：
```
预测结果 + 评估指标 + 特征重要性
    ↓
draw_fit_plot() → 真实值vs预测值对比图
    ↓
draw_feature_importance() → 特征重要性柱状图
    ↓
export_pred_excel() → Excel预测报告
    ↓
export_log() → TXT评估日志
```

## 七、错误处理设计

### 7.1 异常处理策略

| 异常类型 | 处理策略 | 用户提示 |
|----------|----------|----------|
| 文件不存在 | 抛出FileNotFoundError | 请检查数据文件路径是否正确 |
| 文件格式错误 | 抛出ValueError | 数据文件格式不正确，请检查Excel文件 |
| 数据缺失 | 自动剔除并记录日志 | 数据存在缺失，已自动剔除不完整记录 |
| 日期格式错误 | 尝试解析并记录 | 日期格式不正确，已尝试自动转换 |
| 模型训练失败 | 记录详细错误并终止 | 模型训练失败，请检查数据质量 |
