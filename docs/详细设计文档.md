## 一、数据字段定义

### 1.1 SCFI数据字段

SCFI（上海出口集装箱运价指数）数据包含以下字段：

| 字段编号 | 字段名称 | 英文名称 | 数据类型 | 单位 | 说明 |
|----------|----------|----------|----------|------|------|
| F001 | 日期 | date | datetime | - | 数据发布日期，格式YYYY-MM-DD |
| F002 | 综合指数 | scfi_composite | float | 指数点 | SCFI综合运价指数 |
| F003 | 欧洲航线 | scfi_europe | float | USD/TEU | 欧洲航线运价 |
| F004 | 地中海航线 | scfi_mediterranean | float | USD/TEU | 地中海航线运价 |
| F005 | 美西航线 | scfi_us_west | float | USD/FEU | 美西航线运价 |
| F006 | 西非航线 | scfi_west_africa | float | USD/TEU | 西非航线运价 |
| F007 | 南美航线 | scfi_south_america | float | USD/TEU | 南美航线运价 |
| F008 | 美东航线 | scfi_us_east | float | USD/FEU | 美东航线运价 |
| F009 | 东南亚航线 | scfi_southeast_asia | float | USD/TEU | 东南亚航线运价 |
| F010 | 韩国航线 | scfi_korea | float | USD/TEU | 韩国航线运价 |

**数据示例**：

| 日期 | 综合指数 | 欧洲航线 | 地中海航线 | 美西航线 | ... |
|------|----------|----------|------------|----------|-----|
| 2024-01-05 | 1103.52 | 1450.00 | 2180.00 | 2850.00 | ... |

### 1.2 韩国出口指数数据字段

韩国出口景气先行指数数据包含以下字段：

| 字段编号 | 字段名称 | 英文名称 | 数据类型 | 说明 |
|----------|----------|----------|----------|------|
| K001 | 日期 | date | datetime | 数据发布日期，格式YYYY-MM-DD |
| K002 | 韩国出口综合景气指数 | korea_composite | float | 韩国出口整体景气指数 |
| K003 | 对中国出口指数 | korea_to_china | float | 韩国对中国出口景气指数 |
| K004 | 对美国出口指数 | korea_to_us | float | 韩国对美国出口景气指数 |
| K005 | 对欧盟出口指数 | korea_to_eu | float | 韩国对欧盟出口景气指数 |
| K006 | 对东南亚出口指数 | korea_to_sea | float | 韩国对东南亚出口景气指数 |
| K007 | 韩国出口先行指数 | korea_leading | float | 韩国出口先行预测指数 |

**数据示例**：

| 日期 | 韩国出口综合景气指数 | 对中国出口指数 | 对美国出口指数 | ... |
|------|----------------------|----------------|----------------|-----|
| 2024-01-05 | 95.2 | 92.5 | 98.3 | ... |

### 1.3 融合数据字段

融合后的数据集包含原始字段和构造的滞后特征字段：

**原始字段（16个）**：

| 类别 | 字段列表 |
|------|----------|
| 日期字段 | date |
| SCFI字段 | scfi_composite, scfi_europe, scfi_mediterranean, scfi_us_west, scfi_west_africa, scfi_south_america, scfi_us_east, scfi_southeast_asia, scfi_korea |
| 韩国出口字段 | korea_composite, korea_to_china, korea_to_us, korea_to_eu, korea_to_sea, korea_leading |

**滞后特征字段（构造，以韩国出口先行指数为例）**：

| 字段名称 | 说明 | 构造逻辑 |
|----------|------|----------|
| korea_leading_lag1 | 滞后1周韩国出口先行指数 | 取上一周korea_leading值 |
| korea_leading_lag2 | 滞后2周韩国出口先行指数 | 取上两周korea_leading值 |
| korea_leading_lag3 | 滞后3周韩国出口先行指数 | 取上三周korea_leading值 |
| korea_leading_lag4 | 滞后4周韩国出口先行指数 | 取上四周korea_leading值 |

**目标字段（预测标签）**：

| 字段名称 | 说明 | 构造逻辑 |
|----------|------|----------|
| scfi_next | 下周SCFI综合指数 | 取下一周scfi_composite值 |

```python
def clean_data(self, df: pandas.DataFrame) -> pandas.DataFrame:
    """
    数据清洗

    功能:
        1. 处理缺失值（关键字段缺失则删除记录）
        2. 检测并处理异常极值（3σ原则）
        3. 记录清洗日志

    参数:
        df: 待清洗的DataFrame

    返回:
        pandas.DataFrame: 清洗后的DataFrame

        2. 删除关键字段缺失的记录
        3. 对于数值字段，计算均值和标准差
        4. 剔除超出 μ±3σ 范围的异常值
        5. 记录清洗统计信息（删除记录数、异常值数）
        6. 返回清洗后的DataFrame

    算法说明:
        异常值判定标准（3σ原则）:
        - 数值超出 [均值 - 3*标准差, 均值 + 3*标准差] 范围视为异常
    """
```

**方法3：build_lag_feature()**

```python
def build_lag_feature(self, df: pandas.DataFrame) -> pandas.DataFrame:
    """
    构造滞后特征（核心方法）

    功能:
        1. 构造韩国出口指数滞后1-4周特征
        2. 构造SCFI滞后1-4周特征
        3. 构造目标变量（下周SCFI）

    参数:
        df: 融合并清洗后的DataFrame

    返回:
        pandas.DataFrame: 包含滞后特征的数据集

    处理逻辑:
        韩国出口指数滞后特征构造:
        1. 遍历滞后周数 lag_weeks (1, 2, 3, 4)
        2. 对于每个滞后周数n:
           - 创建新列 korea_leading_lag_n
           - 使用 df['korea_leading'].shift(n) 获取滞后n周的值
        3. 对其他韩国出口指标字段重复同样操作

        SCFI滞后特征构造:
        1. 遍历滞后周数 lag_weeks (1, 2, 3, 4)
        2. 对于每个滞后周数n:
           - 创建新列 scfi_composite_lag_n
           - 使用 df['scfi_composite'].shift(n) 获取滞后n周的值

        目标变量构造:
        1. 创建目标列 scfi_next
        2. 使用 df['scfi_composite'].shift(-1) 获取下一周的值

        边界处理:
        1. 删除因滞后特征导致的空值记录（前后各lag_weeks行）
        2. 确保特征和标签对齐

    示例:
        原始数据（korea_leading列）:
        第1周: 100
        第2周: 102
        第3周: 98
        第4周: 105

        构造滞后特征后:
        第3周的 korea_leading_lag1 = 第2周的值 = 102
        第3周的 korea_leading_lag2 = 第1周的值 = 100
    """
```

```python
def calc_evaluation_metrics(self) -> dict:
    """
    计算评估指标

    返回:
        dict: 包含R²、MAE、RMSE等指标

    指标说明:
        R² (决定系数):
        - 公式: R² = 1 - (SS_res / SS_tot)
        - SS_res = Σ(y_true - y_pred)²
        - SS_tot = Σ(y_true - y_mean)²
        - 范围: 0~1，越接近1表示拟合越好

        MAE (平均绝对误差):
        - 公式: MAE = (1/n) * Σ|y_true - y_pred|
        - 含义: 预测值与真实值的平均偏差
        - 单位: 与原数据相同

        RMSE (均方根误差):
        - 公式: RMSE = √[(1/n) * Σ(y_true - y_pred)²]
        - 含义: 预测误差的标准差
        - 单位: 与原数据相同

    处理逻辑:
        from sklearn.metrics import r2_score, mean_absolute_error, mean_squared_error

        metrics = {
            'r2': r2_score(self.y_true, self.y_pred),
            'mae': mean_absolute_error(self.y_true, self.y_pred),
            'rmse': np.sqrt(mean_squared_error(self.y_true, self.y_pred))
        }

        return metrics
    """
```

## 三、核心算法逻辑

### 3.1 时序对齐算法

**算法名称**：双指数日期对齐算法

**算法目标**：将SCFI数据和韩国出口指数数据按日期对齐合并

**输入**：
- SCFI DataFrame（包含date列）
- 韩国出口指数DataFrame（包含date列）

**输出**：
- 融合后的DataFrame（日期完全匹配）

**算法步骤**：

```
步骤1: 格式化日期
    FOR EACH df IN [scfi_df, korea_df]:
        df['date'] = pd.to_datetime(df['date'])
        df = df.sort_values('date')

步骤2: 内连接合并
    fusion_df = pd.merge(scfi_df, korea_df, on='date', how='inner')

步骤3: 重置索引
    fusion_df = fusion_df.reset_index(drop=True)

步骤4: 返回融合结果
    RETURN fusion_df
```

**算法复杂度**：O(n log n)，主要时间消耗在排序

---

### 3.2 滞后特征构造算法

**算法名称**：多阶滞后特征构造算法

**算法目标**：为韩国出口指数和SCFI构造滞后1-4周特征

**输入**：
- 融合后的DataFrame
- 滞后周数配置（lag_weeks=4）

**输出**：
- 包含滞后特征的DataFrame

**算法步骤**：

```
步骤1: 构造韩国出口指数滞后特征
    korea_features = ['korea_composite', 'korea_leading', ...]
    FOR EACH feature IN korea_features:
        FOR lag IN range(1, lag_weeks + 1):
            new_col_name = f'{feature}_lag{lag}'
            df[new_col_name] = df[feature].shift(lag)

步骤2: 构造SCFI滞后特征
    scfi_features = ['scfi_composite']
    FOR EACH feature IN scfi_features:
        FOR lag IN range(1, lag_weeks + 1):
            new_col_name = f'{feature}_lag{lag}'
            df[new_col_name] = df[feature].shift(lag)

步骤3: 构造目标变量
    df['scfi_next'] = df['scfi_composite'].shift(-1)

步骤4: 处理边界缺失值
    df = df.dropna()

步骤5: 返回结果
    RETURN df
```

**算法说明**：
- shift(n) 向后移动n个位置，取前n周的值
- shift(-n) 向前移动n个位置，取后n周的值
- 构造滞后特征后，前后各会缺少lag_weeks行数据

**算法复杂度**：O(n * k)，n为数据行数，k为特征数量

---

### 3.3 模型训练算法

**算法名称**：随机森林回归训练算法

**模型选择**：本系统默认使用随机森林回归模型

**输入**：
- 训练特征X_train
- 训练标签y_train

**输出**：
- 训练完成的随机森林模型

**算法步骤**：

```
步骤1: 初始化模型
    model = RandomForestRegressor(
        n_estimators=100,    # 决策树数量
        max_depth=10,        # 树的最大深度
        min_samples_split=5, # 节点分裂最小样本数
        min_samples_leaf=2,  # 叶节点最小样本数
        random_state=42,     # 随机种子
        n_jobs=-1            # 并行计算
    )

步骤2: 模型训练
    model.fit(X_train, y_train)

步骤3: 特征重要性计算（训练过程中自动计算）
    feature_importances = model.feature_importances_

步骤4: 返回模型
    RETURN model
```

**随机森林原理**：
- 集成多棵决策树，通过Bagging方法降低方差
- 每棵树使用随机子集的特征和样本
- 最终预测为所有树预测值的平均

**超参数说明**：

| 参数 | 说明 | 默认值 |
|------|------|--------|
| n_estimators | 决策树数量 | 100 |
| max_depth | 树的最大深度 | 10 |
| min_samples_split | 节点分裂最小样本数 | 5 |
| min_samples_leaf | 叶节点最小样本数 | 2 |
| random_state | 随机种子 | 42 |

### 3.4 预测推理算法

**算法名称**：下周SCFI预测算法

**输入**：
- 训练完成的模型
- 当周数据（韩国出口指数、SCFI及历史数据）

**输出**：
- 下周SCFI预测值

**算法步骤**：

```
步骤1: 提取当周特征
    current_features = extract_features(current_data)

步骤2: 构造特征向量
    feature_vector = [
        current_data['korea_composite'],
        current_data['korea_leading'],
        current_data['korea_leading_lag1'],
        current_data['korea_leading_lag2'],
        current_data['korea_leading_lag3'],

        current_data['scfi_composite'],

        current_data['scfi_composite_lag2'],
        current_data['scfi_composite_lag3'],
        current_data['scfi_composite_lag4']
    ]

步骤3: 模型预测
    prediction = model.predict([feature_vector])

步骤4: 返回预测值
    RETURN prediction[0]
```

**注意事项**：
- 特征顺序必须与训练时一致
- 缺失的历史数据需要从数据库或文件中获取
- 预测结果为浮点数，表示下周SCFI指数值

### 3.5 数据清洗算法

**算法名称**：时序数据清洗算法

**输入**：
- 融合后的DataFrame

**输出**：
- 清洗后的DataFrame

**算法步骤**：

```
步骤1: 缺失值处理
    key_columns = ['date', 'scfi_composite', 'korea_leading']
    df = df.dropna(subset=key_columns)

步骤2: 异常值检测（3σ原则）
    FOR EACH numeric_column IN df:
        mean = df[column].mean()
        std = df[column].std()
        lower_bound = mean - 3 * std
        upper_bound = mean + 3 * std
        df = df[(df[column] >= lower_bound) & (df[column] <= upper_bound)]

步骤3: 重置索引
    df = df.reset_index(drop=True)

步骤4: 返回清洗结果
    RETURN df
```

**清洗规则说明**：
- 关键字段缺失：直接删除整条记录
- 数值异常：超出±3σ范围视为异常值并剔除
- 日期格式：确保为datetime类型
