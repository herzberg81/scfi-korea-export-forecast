## 一、业务需求

### 1.1 背景与目标

#### 1.1.1 行业背景

集装箱海运是国际贸易的重要组成部分，运价波动直接影响货代企业、船公司的运营成本和利润。传统运价预测主要依赖历史SCFI数据分析，缺乏对外贸先行信号的利用，预测精度有限。

韩国作为全球重要贸易国，其出口景气指数具有明显的先行性特征，能够提前反映国际航运需求变化。将韩国出口先行指数引入海运运价预测，具有重要的行业创新价值。

#### 1.1.2 业务目标

本系统旨在实现以下核心业务目标：

1. **运价预判**：帮助货代企业、船公司根据韩国出口先行景气指标预判下周上海集装箱运价走势
2. **决策支持**：为企业提前锁定舱位、制定合理报价提供数据支撑
3. **风险规避**：通过提前预判运价波动，降低企业运营风险
4. **多航线覆盖**：支持综合指数及8条主要航线运价预测，满足多样化业务需求

### 1.2 用户群体

| 用户类型 | 角色 | 主要需求 |
|----------|------|----------|
| 货代企业 | 运营决策人员 | 提前预判运价走势，制定报价策略 |
| 船公司 | 市场分析人员 | 预测航线运价趋势，优化运力配置 |
| 贸易企业 | 采购物流人员 | 规划出运时机，控制物流成本 |
| 研究机构 | 分析师 | 分析外贸与运价关联关系 |

### 1.3 核心业务场景

**场景一：运价预测决策**

- 货代企业在每周韩国出口指数发布后，需要预判下周运价走势
- 系统自动融合当周韩国出口指数与历史SCFI数据
- 输出下周SCFI综合指数及各航线运价预测值
- 支持企业提前锁定舱位、制定报价

**场景二：多航线运价分析**

- 用户选择特定航线（如欧洲航线、美西航线等）
- 系统输出该航线下周运价预测值
- 支持多条航线对比分析

**场景三：模型精度评估**

- 分析人员评估预测模型精度
- 查看R²、MAE、RMSE等评估指标
- 分析韩国出口指数对运价预测的贡献权重
